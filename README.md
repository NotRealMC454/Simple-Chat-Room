# Simple-Chat-Room

A simple chat room that requires node.js to run and port forwarding (You should use cloudflare or playit.gg to make it public) I am not typing out on how to do it rn tho ask gemini or smth

Run `npm install` or `bun install` to install dependencies
Run `npm start` or `bun start` to start the server
